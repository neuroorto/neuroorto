Наша экосистема NeuroOrto.TS объеденяет множество функций

Это лишь часть ключевых функций нашей экосистемы — чтобы по-настоящему оценить её возможности, вам стоит опробовать её на практике

Автоматический расчет индекса Реймерса

Автоматически рассчитывает индекс миграции (Реймерса) на основании загруженного рентгеновского снимка.

Определение типа ДЦП

Помогает врачу диагностировать тип церебрального паралича по клиническим признакам с подсказками и визуальными примерами.

Оценка по классификации GMFCS

Определяет уровень нарушений по шкале GMFCS (большие моторные функции) с учетом возраста ребенка, предоставляя рекомендации по дальнейшему обследованию и лечению.

График рентгенологических обследований

Формирует индивидуальный график рентгенов таза в зависимости от уровня по GMFCS для оптимального контроля состояния тазобедренных суставов.

Оповещения о риске остеопороза

Выявляет детей в группе риска по остеопорозу и направляет на денситометрию, помогает врачу вовремя направить к эндокринологу.

Подбор ортезов по походке

На основе видео походки и амстердамской классификации приложение помогает определить тип патологической походки и подобрать оптимальный ортез.

Протоколы и маршрутизация

Встроенные клинические протоколы и алгоритмы ведения позволяют врачу быстро принять решение по маршрутизации ребенка.

Рекомендации по ботулинотерапии

На основании типа ДЦП и уровня GMFCS предлагает целесообразность направления на ботулинотерапию.

Родительский помощник

Версия приложения для родителей: напоминания о процедурах, анкеты по симптомам, инструкции по уходу и поддержке ребенка.
